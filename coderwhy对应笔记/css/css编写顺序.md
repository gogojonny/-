# css编写顺序

我认为在编写css的时候，写的时候不需要考虑顺序，只需要在编写完成之后在展示最终的顺序

一，定位：

* position
* display
* float

二，元素展示：

* opacity
* display：none
* visibility

三，盒子模型：

* width、height，background
* margin，padding，box-sizing
* box-shadow

四，文字，文本的样式

* line-height
* text-aline，
* font-size

五，其他：

* transform
* transition
* overflow
* white-space
* 。。。
